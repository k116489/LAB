--使用github的流程--
1.	在github上新增專案(respository)後，複製其ssh網址
2.	複製專案到local端
$ git clone "ssh網址"
$ cd到資料夾
$ git status 查看專案修改狀況
	On branch main
	Your branch is up to date with 'origin/main'.
	nothing to commit, working tree clean
3.	於local端修改專案
隨便在專案中心新增一些內容(例如增加檔案"how to use git.txt")
$ git status 查看專案修改狀況，發現有更改的部分
	On branch main
	Your branch is up to date with 'origin/main'.

	Untracked files:
  	(use "git add <file>..." to include in what will be committed)
        	how to use git.txt(紅字)

	nothing added to commit but untracked files present (use "git add" to track)
$git add "how to use git.txt"
	On branch main
	Your branch is up to date with 'origin/main'.

	Changes to be committed:
  	(use "git restore --staged <file>..." to unstage)
        	new file:   how to use git.txt(綠字)
$git commit -m"任何想要加註的文字" 在local端對即將要commit的檔進行儲存
	[main 2a88f45] try to use git
 	1 file changed, 1 insertion(+)
 	create mode 100644 how to use git.txt
4.	將儲存完的檔案push到global端
